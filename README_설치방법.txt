1️⃣  install.bat 파일을 더블클릭합니다.

2️⃣  "Windows의 PC 보호" 창이 뜨면:
    → "추가 정보" 클릭
    → "실행" 클릭

3️⃣  설치가 완료되면 바탕화면에 
    "샤인 주차별 전문가" 아이콘이 생성됩니다.

4️⃣  이제 바탕화면 아이콘으로 바로 실행하세요!

💡 사용 팁
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 열가둠 시작일과 분석일을 설정하면 현재 주차가 자동 계산됩니다.
• 각 주차를 클릭하면 상세 재배 정보를 확인할 수 있습니다.
• 작업 체크리스트는 브라우저에 저장되어 유지됩니다.
• 날씨 정보는 인터넷 연결 시 자동으로 업데이트됩니다.
